# 🌾 Crop Prediction System using Machine Learning (Flask App)

## 📌 Overview

This project is a Machine Learning web application that predicts the best crop to grow based on soil and environmental conditions like Nitrogen, Phosphorus, Potassium, Temperature, Humidity, pH, and Rainfall.

The model is built using **Random Forest Classifier** and deployed using **Flask**.



## 🚀 Features

* Predicts suitable crops based on input values
* Simple and user-friendly UI
* Machine Learning model integration
* Flask-based web application



## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas
* NumPy
* HTML & CSS

## 📊 Model Details

* Algorithm: Random Forest Classifier
* Input Features:

  * Nitrogen
  * Phosphorus
  * Potassium
  * Temperature
  * Humidity
  * pH
  * Rainfall
